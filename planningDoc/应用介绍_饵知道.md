# 饵知道 - 智能钓鱼配方助手

> 智能配方 · 记录渔获 · 让每一次出钓都有收获

---

## 📱 应用简介

**饵知道**是一款专为钓鱼爱好者打造的智能饵料配方助手应用。基于HarmonyOS NEXT开发，结合实时天气、地理位置和丰富的鱼类习性数据库，为钓友提供科学、精准的饵料配方推荐，帮助您在不同环境下都能找到最合适的饵料搭配。

无论您是刚入门的新手钓友，还是经验丰富的老钓手，饵知道都能成为您的得力助手，让钓鱼变得更简单、更有趣、更有收获！

---

## ✨ 核心功能

### 🎯 智能配方生成

**一键生成专属配方**
- 选择目标鱼种、水域类型、季节天气
- 智能引擎分析环境因素，生成最优配方
- 支持快速生成和精确匹配两种模式
- 配方包含详细的原料配比、使用建议

**精确匹配模式**
- 12项环境参数精细调节
- 五维能力分析（诱鱼性、适口性、状态、留鱼性、性价比）
- 匹配度评分系统（0-100分）
- 智能反馈调整，不满意可重新生成

### 📚 丰富配方库

**30+经典配方**
- 覆盖鲫鱼、鲤鱼、草鱼、鳙鱼、鲢鱼等常见鱼种
- 适配池塘、河流、水库、湖泊等多种水域
- 春夏秋冬四季配方全覆盖
- 详细的原料清单、制作步骤、使用技巧

**自定义配方**
- 支持创建个人专属配方
- 自定义原料配比和味型
- 状态特性可视化调节
- 配方收藏和历史记录

### 🗺️ 钓点地图

**周边钓点发现**
- 基于华为Petal Maps的精准定位
- 自动搜索周边5公里内钓点
- 显示钓点类型（池塘、河流、水库等）
- 点击钓点推荐适合的饵料

**逆地理编码**
- 点击地图任意位置获取地名
- 精确到区/街道级别
- 方便记录钓点位置

### 📝 渔获记录

**完整记录每次出钓**
- 记录鱼种、重量、数量
- 支持添加多张渔获照片
- 自动获取天气和位置信息
- 可关联使用的配方

**数据统计分析**
- 总渔获重量统计
- 出钓次数记录
- 最大单尾记录
- 按鱼种、钓点、日期筛选

### 🌤️ 实时天气

**和风天气API集成**
- 实时温度、天气状况
- 风向风力信息
- 气压数据（影响鱼口）
- 智能钓鱼建议

### 👤 用户等级系统

**成长激励机制**
- 5个等级：新手钓友 → 入门钓手 → 进阶钓客 → 资深钓友 → 钓鱼大师
- 经验值计算：渔获重量×10 + 出钓次数×20 + 钓获尾数×5
- 等级进度可视化展示
- 解锁专属称号和图标
